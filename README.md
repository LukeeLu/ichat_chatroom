# iChat聊天室

### 一款基于socket通信的聊天软件

## 功能特点   
- [] 前后端分离聊天室      
- [] 页面优雅简洁
- [] 双端加密通信
- [] 群聊功能
-   []  暗黑模式
-  [X] 离线消息
-  [X] 消息备份
-  [X] 发送文件
