# iChat聊天室

### 一款基于socket通信的聊天软件

## 功能特点   
- [X] 前后端分离聊天室      
- [X] 页面优雅简洁
- [X] 双端加密通信
- [X] 群聊功能
-   [X]  暗黑模式
-  [] 离线消息
-  [] 消息备份
-  [] 发送文件
